# Plantilla Marketing de Contenido — Sistema Completo

Este skill documenta la plantilla completa de marketing de contenido alojada en Notion.
La plantilla es un sistema integral para planificar, crear y distribuir contenido estratégico.

## Estructura General de la Plantilla

La plantilla tiene estos módulos principales:

1. **Estrategia de contenido** (página central)
   - Audiencia (Investigación + Insights)
   - Canales
   - Contenido (Ángulos de autoridad + Topic Clusters)
   - Tono, voz y estilo
   - Posicionamiento de producto o servicio

2. **Content Calendar** (base de datos operativa)

3. **Recursos** (galería con subpáginas)
   - AI tools
   - Benchmarks
   - Prompts
   - Principios psicológicos aplicados a la creación de contenido

4. **Herramientas recomendadas** (stack tecnológico)

5. **Framework visual** (mapa completo del sistema)

Para el contenido detallado de cada módulo, leer: `references/modulos-detallados.md`
Para el generador de hooks completo, leer: `references/generador-hooks.md`
Para el framework de investigación de audiencias, leer: `references/investigacion-audiencias.md`

## Flujo de Trabajo Recomendado

Cuando se inicia un proyecto nuevo para un cliente, seguir este orden:

### Paso 1: Audiencia (JTBD + Insights)
- Completar la tabla de Audiencia Insights con: Name, Situación, Problema, Miedo, Motivación/Deseos
- Aplicar el framework de Investigación de audiencias (Jobs to be Done)
- Identificar audiencia primaria, secundaria y terciaria

### Paso 2: Posicionamiento
- Definir el posicionamiento del producto/servicio
- Establecer diferenciadores claros vs competencia
- Definir propuesta de valor única

### Paso 3: Tono, Voz y Estilo
- Definir la personalidad de marca en comunicación
- Establecer guías de lenguaje para cada plataforma

### Paso 4: Topic Clusters
- Crear clusters temáticos con 4 sub-temas cada uno:
  - Sub-tema 1: Producto/PLG (contenido centrado en el producto)
  - Sub-tema 2: Contenido como eje (educativo/editorial)
  - Sub-tema 3: SEO/canales orgánicos
  - Sub-tema 4: Data (métricas, análisis)

### Paso 5: Canales
- Seleccionar canales usando el checklist de la plantilla
- Categorizar en: Propios, Social, Búsqueda, Terceros
- Crear plan de implementación

### Paso 6: Hooks y Ángulos
- Aplicar los 7 principios psicológicos del Generador de Hooks
- Crear ángulos de contenido de autoridad

### Paso 7: Content Calendar
- Crear entradas en la base de datos con los campos requeridos
- Asignar estados, owners, fechas y plataformas

## Content Calendar — Campos de la Base de Datos

La base de datos del Content Calendar tiene estos campos:
- **Nombre de tu idea**: título descriptivo del contenido
- **Topic clusters**: a qué cluster pertenece
- **Tipo de contenido**: formato (video corto horizontal, carrusel, post, artículo, etc.)
- **Plataforma**: dónde se publica (LinkedIn, Instagram, TikTok, etc.)
- **Owner**: persona responsable
- **Status**: estado del contenido (Idea, En producción, Revisión, Publicada, etc.)
- **Fecha de creación**: cuándo se creó la idea
- **Fecha de publicación**: cuándo se publica
- **Post URL**: enlace al contenido publicado
- **Archivos**: assets asociados

Vistas disponibles: Contenido (tabla), Calendario, Por estado, y 3 más.

## Framework Visual — Customer Journey

El framework mapea contenido al customer journey con estas etapas:

### Awareness (Conocimiento)
- Formatos B2B: Investigación original, Framework/libros de opinión, Podcast/prensa, Artículos de blog, Webinars
- Formatos B2C: Artículos/blogs, Videos, Infografías

### Consideración
- B2B: Emails interactivos, Formatos de producto/prueba, Calculadora ROI, Templates/recursos, Webinars de profundización
- B2C: Guías de compra, Comparativas, Testimonios

### Conversión
- B2B: Competitivos/ofertas, Términos financiamiento, Guías financiamiento
- B2C: Estudios de caso, Demos, Ofertas especiales

### Retención y Expansión
- B2B: Templates exclusivos, Email series, Comunidad, Referidos, UGC
- B2C: Newsletters, Programas de fidelidad, Referidos
